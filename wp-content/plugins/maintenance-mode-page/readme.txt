== Installation ==

1. Upload the entire `maintenance-mode-page` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Maintenance Mode Panels menu to customize this plugin.
4. Finally Enjoy it.
